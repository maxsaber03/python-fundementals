# Python Fundamentals – Coursework Portfolio

This repository contains structured Python exercises demonstrating foundational programming concepts.

Core Areas:

- String manipulation and transformation  
- Basic parsing logic and expression evaluation  
- File I/O and data processing  

These implementations were completed as part of coursework and later reorganized for clarity and presentation.

Topics Demonstrated:

- Caesar cipher (character shifting logic)
- Sentence formatting and normalization
- Custom string-based math parser
- File-based grade average computation

Focus: Clear logic, control flow, and disciplined problem solving.
